### shortcut
~~~~
ctrl+enter : run
ctrl+space : layer toggle
ctrl+esc : abort
~~~~

### game on key
~~~~
f : decide
d : cancel
s : Y key
a : X key
e : previus
r : next
arrow-up : up
arrow-down : down
arrow-left : left
arrow-right : right
start : space
~~~~
