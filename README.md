### quest
dragon quest macro for monospaced art game

### package
```
//pug
script(src="https://gnjo.github.io/quest.js")
```

### group
```
//pug
script(src="https://gnjo.github.io/underscore.js")

```

>>> //:jump to or jump back

LAY //:X00...XZZ XXX to LAY
MAP //:A00...AZZ to MAP
DAT //:%00...%ZZ to DAT
